Wat zijn rest parameters? Geef een voorbeeld.
Wat is het verschil tussen rest parameters en expliciete parameters die je aangeeft bij een functie?

parameters kun je als volgt weergeven in een functie:
let sum = (a,b) => a + b ;

rest-parameters zijn parameters die je geen naam hebt gegeven.

Drie verschillen tussen rest parameters en expliciete parameters:

  • rest parameters krijgen als enige geen aparte naam (i.e. formeel gedefinieerd in functie), terwijl het arguments object alle arguments bevat die aan de functie worden meegegeven;
  • het arguments object is geen echte array, rest parameters zijn Array instances, wat betekent dat methoden als sort, map, forEach of pop direct kunnen worden toegepast;
  • het arguments object heeft extra eigenschappen .

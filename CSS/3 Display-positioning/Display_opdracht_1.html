Position properties

Er zijn de volgende position properties te onderscheiden:

  1. static: default. Elementen staan in volgorde
  2. absolute: het element is verwijderd van de flow van het document en alle andere Elementen gedragen zich alsof het er niet is.
  3. fixed: verwijderd van flow document; relative tot document, element staat vast, ook al scroll je
  4. relative: hiermee nudge/manipuleer je het element weg van de locatie waarop het stond:
  5. sticky: heeft relative waarde tot je op een bepaald punt komt waar een element fexed wordt
  6. initial
  7. inherit: position value cascade niet dus hiermee kan het wel, zodat de eigenschappen van de parent gekregen worden

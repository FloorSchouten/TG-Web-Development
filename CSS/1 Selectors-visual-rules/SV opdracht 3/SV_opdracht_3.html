Er zijn verschillende selectors, bijvoorbeeld; ● type selectors ● descendant selectors ● class selectors Maak van de bovengenoemde selectors voorbeelden. Geef in je antwoord ook aan wat de voordelen zijn van elke selector. Type selector: zorgt dat alle elementen van een bepaald type binnen het document worden geselecteerd. Voordeel: handig, omdat het snel werkt en er geen onnodige conflicten of afwijkingen ontstaan per type. Bijvoorbeeld: je wilt alle paragrafen definieren. element {style properties} p { color: purple; } Descendant selectors: een van de CSS combinator selectors. Geeft een relatie aan van alle elementen die een descendant (afstammeling) zijn van een bepaald element. Bijvoorbeeld als alle p: red, dan kun je uitzonderingen maken door een div te maken waarin bepaalde paragrafen een andere kleur krijgen. Het voordeel is dat je binnen een bepaald type uitzonderingen kunt maken Bijvoorbeeld: p { color: purple } div p { background-color: red } Class selectors: combineert elementen van een bepaalde class; definieert dan alle elementen die rood van kleur zijn, of

  • elementen die rood van kleur zijn en een grijze achtergrond hebben. Bijvoorbeeld: .red p{ color: #f33 }

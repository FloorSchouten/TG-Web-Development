Opdracht 1. Welke 3 manieren kun je css inladen?

CSS kan worden ingeladen dmv een extern CSS sheet; eindigt op .css. Alle styling staat hierop, je maakt een link in de head van het document.

CSS kan ook intern worden ingeladen dmv css informatie in het style, het head element. Dit doe je als een pagina een unieke layout heeft.

CSS kan ook inline worden ingevoegd, als een element een bijzondere layout heeft.

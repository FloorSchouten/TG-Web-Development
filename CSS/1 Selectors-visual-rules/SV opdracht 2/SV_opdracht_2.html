Opdracht 2. Wat betekent de syntax van CSS?

Een syntax betekent de manier waarop de regels in CSS zijn opgebouwd. Een regel wordt opgebouwd door een selector en een declaration block. De selector verwijst naar het element dat je van een stijl wilt voorzien. Een declaration block bevat een of meer definities van de selector, zoals kleur en grootte. Begint en eindigt met een {}; definitie bestaat uit property name:value; Na iedere definitie een puntkomma Het ziet er dan zo uit: p { color: red; text-align: center; }

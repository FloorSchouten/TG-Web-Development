Kun je enkele voorbeelden geven van pseudo-elementen (minimaal 4)? Beschrijf met je eigen woorden wat deze pseudo-elementen inhouden.

::after: key word dat wordt toegevoegd aan een selector om een gedeelte van de geselecteerde elementen te stylen. wordt weergegeven na het aangegeven element

::first-letter: geeft stijl aan de eerste letter van een block-level element maar alleen als er geen image of table aan vooraf gaat

::before: selecteert elementen die first child zijn van een element en geven daar stijl aan. Bijvoorbeeld om een hartje toe te voegen aan een link

::placeholder: representeert de placeholder tekst op een plaats waar input ingevuld moet worden
